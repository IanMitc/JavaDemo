# CSS

CSS stands for cascading style sheet
and primary use for styling our web page

- css can be describe in 3 ways
1. external css
2. internal css
3. inline css

# external css
It is use to maintain external .css file which will have all common styling for every page

css/style.css

<head>
  <link href="style.css" rel="stylesheet"/>
</head>


# internal css 
It is use to have exclusive css for a particular web page and can be describe within a HTML page

<head>
  <style>
	// css code
  </style>
</head>

# inline css
It is css which we can describe while declare the html elements

<h1 style="css code">Heading</h1>
